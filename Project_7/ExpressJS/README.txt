
api.js contains all the APIs that are used in the project as a whole. The APIs are as follows:

1. Search all the records 

2. Search by zonal_area

3. Search by City

4. Search by name

5. Search by zonal_area and name

6.Search by city and name

7.Search by zonal_area,city and name

8.Search all by using projections and sort by ratings

9.Search by zonal_area and sort by ratings

10.Search by city and sort by ratings

11.Search by name and sort by ratings

12.Search by zonal_area and city and sort by ratings

13.Search by zonal_area and name and sort by ratings

14.Search by city and name and sort by ratings

15.Search by zonal_area, city and name and sort by ratings
